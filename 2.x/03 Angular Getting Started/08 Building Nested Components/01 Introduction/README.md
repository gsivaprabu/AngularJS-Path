# Angular Getting Started - 08 Building Nested Components - 01 Introduction

- Components designed to be nested with other components.
- Establish communication between components
- We can display the component as directive.


### What makes a component Nest-able ?
- Its template only manages a fragment of a large view
- It has a selector  
- It optionally communicates with its container.
